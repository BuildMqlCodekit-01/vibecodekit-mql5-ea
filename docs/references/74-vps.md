# MetaQuotes Native VPS

MetaQuotes' VPS rents a remote terminal that mirrors your
local broker login.  `/mql5-deploy-vps` emits a `MIGRATE-VPS.md`
checklist covering the manual migration steps (you cannot script the
VPS provisioning — it's behind the broker login).
